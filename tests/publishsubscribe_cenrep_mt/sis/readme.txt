Creating a sis that installs the module test and its cenrep key to device:
  1. Create and sign mt_configurability_cenrep.sis from mt_configurability_cenrep.pkg.
  2. Create and sign moduletest_configurability.sis from moduletest_configurability.pkg.

Only moduletest_configurability.sis should be installed in device. The cenrep sis file is embedded inside the final sis.
